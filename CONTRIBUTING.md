# Contributing to Blip Engine Arcade

Pull requests are welcome at [github.com/jacobandresen/blip](https://github.com/jacobandresen/blip).

The `blip` library handles the [Macroquad](https://macroquad.rs/) and Rust plumbing so you can focus on game logic.

## Adding a Game

The Golden Age of Arcade Video Games (roughly 1978–1986) produced hundreds of titles worth revisiting — Asteroids, Centipede, Frogger, Pac-Man, Donkey Kong, Tempest, and many more. That era is a deep well of inspiration.

## Copyright Guidelines

Please honour copyrights when contributing:

- Reimplement mechanics and aesthetics from scratch — do not copy proprietary code, graphics, or audio.
- Original names are trademarked; use alternative titles as this project does (Rally instead of Pong, Serpent instead of Snake, and so on).
- When in doubt, treat your implementation as a tribute, not a reproduction.
